Converter PREÇO REAL para PREÇO ESTIMADO ?

Opção ideal para quem deseja transformar a última lista de compra realizada em lista de compras modelo e ainda salvar o PREÇO REAL como estimado para melhor monitoramento. Nota: Apenas serão invertido os preços no quais os produtos foram comprados e o preço real seja maior que zero.

NÃO

Como usar?

Sem necessidade de um app. Lista de compras desenvolvida com intuito do usário fazer uma pré-lista com preço estimado mesmo ainda em casa para assim, no mercado ter a oportunidade de fazer um comparativo entre valores com o valor real, assim possibilitando maior controle sobre seus gastos.

A lista fica salva ao fechar o navegador ?

Sim!! Não há problema de fechar o navegador, pois a lista é salva no cache do navegador, assim que o usuário acessar novamente o site a lista é recuperada. Caso o usuário não deseje mais a lista ou nada mais salvo no navegador, basta excluir todos os produtos da lista que o o cache é limpado automaticamente.

Como não esquecer o link do site ?

Favorite o site para não perde-lo de vista e esteja bem fácil o seu acesso!! Site ideal e você não precisa sujar seu aparelho instalando nenhum aplicativo!!!
